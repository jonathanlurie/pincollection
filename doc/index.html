addPin

Add a pin to the collection. This pin can have an image or be a circle, or a star

addPin(position: Object, options: Object)
Parameters
position (Object) 3D coordinates such as {x: Number, y: Number, z: Number}
options (Object = {}) the option object
Name Description
options.id string the unique ID of this pin within the collection. (default: auto generated)
options.makeVisible boolean whether to make this pin visible or not by default (default: true)
options.color String hex string color of the pin (default: '#ffffff')
options.size Number size (diameter of a circle) of the pin (default: 1000)
options.textureUrl string URL of an image (png or jpg) to load on the pin (default: null)
options.shape String shape of the pin. Can be 'circle' or 'star'. Not compatible with options.textureUrl (default: circle)
options.sides Number only with options.shape is 'star'. Number of branches in the star (default: 16)
options.spikiness Number only with options.shape to 'star'. In range [0, 1] , very spiky when 0, very not spiky when 1. (default: 0.3)
options.onDoubleClick Function Callback function called when the pin is double-clicked. Will be called with the args: (pinID: String, pinPosition: {x: Number, y: Number, z: Number}, dblclickMouseEvent: Object)
options.onMouseMove Function Callback function called when the pin is hovered. Will be called with the args: (pinID: String, pinPosition: {x: Number, y: Number, z: Number}, dblclickMouseEvent: Object)
options.onLeave Function Callback function called when the pointer leaves the pin. Will be called with the args: (pinID: String, dblclickMouseEvent: Object)
